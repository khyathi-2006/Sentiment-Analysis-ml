# Sentiment Analysis using Machine Learning

This project performs sentiment analysis on text using Python and NLP techniques.

## Technologies Used
- Python
- Pandas
- Numpy
- Scikit-learn
- NLTK
- TF-IDF
- Logistic Regression

## Features
- Text preprocessing
- Stopword removal
- TF-IDF vectorization
- Machine learning model
- Accuracy evaluation
- User input sentiment prediction

## Dataset
Custom dataset with positive and negative sentences.

## How to Run

Install dependencies:

pip install -r requirements.txt

Run project:

python sentiment.py

## Example

Input:
I love this product

Output:
Positive
